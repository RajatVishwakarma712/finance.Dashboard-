# 📊 Investment Survey Analysis Dashboard – Power BI

## Overview

This Power BI dashboard is developed using an Investment Survey dataset. The objective of this project is to analyze people's investment behavior, preferences, financial goals, and investment awareness through interactive visualizations. The dashboard helps users understand how different demographic groups make investment decisions and which investment avenues are most preferred.

---

# Dataset Description

The dataset contains survey responses from individuals regarding their investment choices, objectives, and financial preferences.

---

# Important Columns Used in the Dashboard

## 1. Gender

Represents the gender of the respondent. This column is useful for comparing investment behavior between different genders.

**Purpose**

* Gender-wise investment analysis
* Comparison of financial preferences

---

## 2. Age

Shows the age of each respondent.

**Purpose**

* Age group analysis
* Understanding investment patterns across different age categories
* Identifying the most active investment age group

---

## 3. Investment_Avenues

Indicates whether the respondent prefers investing or not.

**Purpose**

* Overall investment participation
* Investment awareness analysis

---

## 4. Mutual_Funds

Represents the respondent's preference for Mutual Funds.

**Purpose**

* Popularity of Mutual Funds
* Comparison with other investment options

---

## 5. Equity_Market

**Purpose**

# Dashboard Objective

The Power BI dashboard provides valuable insights into investment behavior by analyzing demographic information, investment preferences, savings objectives, expected returns, investment duration, and decision-making factors. Interactive visualizations enable users to compare investment options, identify trends, and better understand how different groups approach financial planning and investment decisions.

---

# Conclusion

This dashboard transforms raw survey data into meaningful business insights using Power BI. It helps users understand investor behavior, identify popular investment avenues, analyze financial objectives, and support data-driven decision-making through clear and interactive visualizations.
